How to make a git commitment/ Start local repository

  • GO to folder in command line and type "git init", this will create a local repository
  • Type "git status" if you want to check the status of the local repository
  • Type "git add" + file name or "git add ." this will add all files to staging area.
  • You can check status with "git status" to see if anything has been added or changes that need to be commited.
  • Type "git commit -m "First Commit" to make your first commit.

How to flip back to previous commits

  • Go to folder and use "git add -m "second commit" You must use git add everytime before you commit.
  • If you contintue to change the document every so often continue to commit.
  • type "git log --oneline" this will show all your commit messages. This will show commit messages and a hash( number in front of messge)
  • type "git checkout ##### index.html (#### stand for unique hash), this will revert the code on the code editor directly to that previous commit.
  • You can jump between commits, the latest commit is called the Head.
  • To go back to the hed type "git reset HEAD index.html" (index.html is the file name we are working on)
  • Aferwards type "git checkout --index.html" this will change the code editor to where the code was at the latest commit.

  • in the line after that add the line "lite": "lite-server" with no comma, add a comma after the first line
